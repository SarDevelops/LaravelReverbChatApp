## Steps to Run Project
- Clone Project Using Command  
    - git clone https://github.com/SarDevelops/LaravelReverbChatApp.git

## Commands 
- Composer Update
- php artisan key:generate
- php artisan migrate --seed
- npm install
- npm run dev
